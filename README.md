# shunting-yard
Shunting yard algorithm and rpn solver.

### Example

```
< node main 'min((4 * 345), 5 ^ sqrt(2))'
> RPN: 4,345,*,5,2,sqrt,^,min
> RESULT: 9.738517742335421
```
